## Tech Stack

- Pure HTML + CSS + Vanilla JavaScript — no frameworks, no dependencies
- Fonts: **DM Sans** + **DM Serif Display** (Google Fonts)
- WhatsApp integration via `wa.me` API

---

## Features

- Fully responsive — mobile, tablet, desktop
- Hamburger menu on mobile
- WhatsApp contact form — form data is pre-filled into a WhatsApp message sent to the business number
- Scroll reveal animations
- Sticky navigation
- Sections: Hero, Services, Why Us, Process, Contact, Footer

---

## How to Run

Just open `index.html` in any browser — no build step, no server needed.

```bash
open index.html
```

---

## WhatsApp Form

When a visitor fills and submits the contact form, it opens WhatsApp with a pre-filled message containing:
- Name
- Phone (if provided)
- Email
- Service of interest (if selected)
- Message (if provided)

Messages are sent directly to **+91 98111 36053**.

---

## Deployment

This is a static site. It can be hosted on:

- **Netlify** — drag and drop the `index.html` file
- **Vercel** — import the folder
- **GitHub Pages** — push to a repo and enable Pages
- Any web hosting that supports static files
